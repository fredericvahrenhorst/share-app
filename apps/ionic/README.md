# Wauz Miau Ionic App

## Setup

```bash
npm install -g @ionic/cli
npm install
```

### Commands

Start Ionic

```bash
ionic serve
```

### VS Code Extensions

Following Extensions are recommended

- EditorConfig for VS Code
- ESLint
- Tailwind CSS IntelliSense
- vscode nvm switcher

### Build iOs

- ionic cap build ios
- xcode automatic start
